# splitter.sh
- for **samsung gear 360, aug 2016**
  - ```ffmpeg required```
  - ```chmod +x splitter.sh```
  - ``` ./splitter.sh```

Problem: Your awesome little Samsung Gear VR Camera records all videos like this: 

C88888D C88888D C8888                                                      
8  .d88b.   .d88b.  8           
8 .8P  88. .8P  88. 8           
8 88  d'88 88  d'88 8           
  88 d' 88 88 d' 88             
8 `88  d8' `88  d8' 8           
8  `Y88P'   `Y88P'  8                                                   
C88888D C88888D C8888   

That's annoying, because autopano needs them to be like this: 

C88888D C88   888D C88888                                                      
8  .d88b. 8   8 .d88b.  8           
8 .8P  88.8   8.8P  88. 8           
8 88  d'888   888  d'88 8           
  88 d' 888   888 d' 88 8            
8 `88  d8'8   8`88  d8' 8           
8  `Y88P' 8   8 `Y88P'  8                                                   
C88888D C88   888D C88888       

Solution: 

Splitter!
